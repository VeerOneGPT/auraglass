### GlassLineChart

Line chart visualization.

```tsx
<GlassLineChart
  data={[
    { x: 1, y: 10 },
    { x: 2, y: 20 },
    { x: 3, y: 15 }
  ]}
  width={400}
  height={300}
/>
```

**Props:**
- `data?: ChartDataPoint[]` - Chart data points
- `width?: number` - Chart width
- `height?: number` - Chart height
