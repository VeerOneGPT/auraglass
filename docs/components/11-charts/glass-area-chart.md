### GlassAreaChart

Area chart visualization.

```tsx
<GlassAreaChart
  data={[
    { x: 'Jan', y: 100 },
    { x: 'Feb', y: 200 },
    { x: 'Mar', y: 150 }
  ]}
  width={400}
  height={300}
/>
```

**Props:**
- `data?: ChartDataPoint[]` - Chart data points
- `width?: number` - Chart width
- `height?: number` - Chart height
