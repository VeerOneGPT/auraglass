### GlassBarChart

Bar chart visualization.

```tsx
<GlassBarChart
  data={[
    { x: 'A', y: 10 },
    { x: 'B', y: 20 },
    { x: 'C', y: 15 }
  ]}
  width={400}
  height={300}
/>
```

**Props:**
- `data?: ChartDataPoint[]` - Chart data points
- `width?: number` - Chart width
- `height?: number` - Chart height
