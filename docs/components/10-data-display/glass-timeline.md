### GlassTimeline

Chronological event timeline.

```tsx
<GlassTimeline
  items={[
    {
      id: '1',
      title: 'Project Started',
      subtitle: 'Initial setup completed',
      time: '2 hours ago',
      icon: <ProjectIcon />
    }
  ]}
/>
```

**Props:**
- `items?: TimelineItem[]` - Timeline items
