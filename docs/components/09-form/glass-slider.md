### GlassSlider

Range slider input.

```tsx
<GlassSlider
  value={value}
  onChange={setValue}
  min={0}
  max={100}
  step={1}
/>
```

**Props:**
- `value?: number` - Current value
- `onChange?: (value: number) => void` - Change handler
- `min?: number` - Minimum value
- `max?: number` - Maximum value
- `step?: number` - Step increment
