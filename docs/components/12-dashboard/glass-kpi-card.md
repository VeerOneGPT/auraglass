### GlassKPICard

KPI metric display card.

```tsx
<GlassKPICard
  title="Revenue"
  value="$125,000"
  change={12.5}
  changeLabel="vs last month"
/>
```

**Props:**
- `title: string` - KPI title
- `value: string | number` - KPI value
- `change?: number` - Percentage change
- `changeLabel?: string` - Change description
