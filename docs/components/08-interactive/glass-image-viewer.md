### GlassImageViewer

Image viewer with zoom and pan controls.

```tsx
<GlassImageViewer
  src="image.jpg"
  alt="Image description"
  zoomable
  pannable
/>
```

**Props:**
- `src?: string` - Image source
- `alt?: string` - Alt text
- `zoomable?: boolean` - Enable zoom
- `pannable?: boolean` - Enable pan
